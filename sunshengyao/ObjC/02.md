# ObjC 语法 #2：Foundation 数据类型

> ObjC 不用 STL，用的是 Foundation 框架。核心就几个类，掌握它们能读 90% 的业务代码。

---

## 1. NSString — 不可变字符串

```objc
// 创建
NSString *s1 = @"Hello";                          // 字面量（最常用）
NSString *s2 = [NSString stringWithFormat:@"%d", 42];  // 格式化
NSString *s3 = [[NSString alloc] initWithUTF8String:"c_str"];  // 从 C 字符串

// 常用操作
[s1 length];                    // 长度（C++: s.size()）
[s1 isEqualToString:s2];        // 比较相等（C++: s1 == s2）⚠️ 不要用 ==
[s1 hasPrefix:@"Hel"];          // 前缀匹配（C++: s.starts_with()）
[s1 hasSuffix:@"lo"];           // 后缀匹配
[s1 uppercaseString];           // 转大写
[s1 lowercaseString];           // 转小写
[s1 substringToIndex:3];        // 取前 3 个字符
[s1 substringFromIndex:2];      // 从第 2 个取到末尾
[s1 substringWithRange:NSMakeRange(1, 3)];  // 从位置 1 取 3 个字符

// 拼接
NSString *full = [s1 stringByAppendingString:@" World"];  // C++: s1 + " World"
NSString *fmt = [NSString stringWithFormat:@"%@ is %d", s1, 42];  // %@ 打印对象

// 查找与替换
NSRange range = [s1 rangeOfString:@"ell"];  // 返回 {location, length}，找不到 length=0
NSString *replaced = [s1 stringByReplacingOccurrencesOfString:@"l" withString:@"L"];

// 分割
NSArray *parts = [@"a,b,c" componentsSeparatedByString:@","];  // C++: split()

// C++ 互转
const char *cStr = [s1 UTF8String];               // → const char*（临时指针，不要持有）
NSString *fromC = [NSString stringWithUTF8String:cStr];  // C 字符串 → NSString
```

### NSMutableString — 可变字符串

```objc
NSMutableString *ms = [NSMutableString stringWithCapacity:100];
[ms appendString:@"Hello"];        // 追加
[ms appendFormat:@" %d", 42];      // 格式化追加
[ms insertString:@"X" atIndex:2];  // 插入
[ms deleteCharactersInRange:NSMakeRange(0, 2)];  // 删除
```

> **实际业务中很少用 NSMutableString**，大多数场景直接 `NSString` + `stringWithFormat` 拼接。可变字符串主要用于大量拼接性能优化。

## 2. NSArray / NSMutableArray — 数组

```objc
// 创建
NSArray *arr = @[@"a", @"b", @"c"];                 // 字面量
NSArray *arr2 = [NSArray arrayWithObjects:@"a", @"b", nil];  // nil 结尾（老写法）
// ⚠️ 字面量 @[] 不能放 nil，老写法 nil 是结束标记

// 访问
[arr objectAtIndex:0];          // 等价于 arr[0]
arr[0];                          // 下标语法（推荐）
[arr firstObject];               // 首元素（空数组返回 nil，不崩）
[arr lastObject];                // 末元素
[arr count];                     // 长度（C++: v.size()）

// 遍历
for (NSString *s in arr) {       // 快速枚举（推荐）
    NSLog(@"%@", s);
}
[arr enumerateObjectsUsingBlock:^(id obj, NSUInteger idx, BOOL *stop) {
    // block 遍历，*stop = YES 可提前终止
}];

// 查找
[arr containsObject:@"a"];                     // 是否包含
NSUInteger idx = [arr indexOfObject:@"b"];       // 找索引，找不到返回 NSNotFound

// 排序
NSArray *sorted = [arr sortedArrayUsingSelector:@selector(compare:)];  // 字符串排序
NSArray *sorted2 = [arr sortedArrayUsingComparator:^NSComparisonResult(id a, id b) {
    return [a compare:b];  // block 排序
}];
```

### NSMutableArray

```objc
NSMutableArray *mArr = [NSMutableArray array];  // 空可变数组
NSMutableArray *mArr2 = [@[@"a"] mutableCopy];   // 从不可变复制

[mArr addObject:@"new"];           // 末尾添加（C++: v.push_back()）
[mArr insertObject:@"x" atIndex:0]; // 指定位置插入
[mArr removeObjectAtIndex:0];       // 按索引删除
[mArr removeObject:@"a"];           // 按值删除
[mArr replaceObjectAtIndex:0 withObject:@"y"];  // 替换
[mArr removeAllObjects];           // 清空

// 排序（原地）
[mArr sortUsingSelector:@selector(compare:)];
```

> **C++ 类比**：`NSArray` ≈ `std::vector<const T*>`（存对象指针，不可变），`NSMutableArray` ≈ `std::vector<T*>`（可变）。

## 3. NSDictionary / NSMutableDictionary — 字典

```objc
// 创建
NSDictionary *dict = @{@"key1": @"val1", @"key2": @42};   // 字面量
NSDictionary *dict2 = [NSDictionary dictionaryWithObjectsAndKeys:@"val1", @"key1", nil];  // 老写法

// 访问
dict[@"key1"];                  // 下标语法（推荐）
[dict objectForKey:@"key1"];    // 等价
[dict allKeys];                 // 所有 key
[dict allValues];               // 所有 value
dict.count;                     // 键值对数量

// 遍历
[dict enumerateKeysAndObjectsUsingBlock:^(id key, id obj, BOOL *stop) {
    NSLog(@"%@ = %@", key, obj);
}];

// 安全取值
NSString *val = dict[@"nonexistent"];  // 返回 nil，不崩（C++ map 会插入默认值）
```

### NSMutableDictionary

```objc
NSMutableDictionary *mDict = [NSMutableDictionary dictionary];
mDict[@"key"] = @"value";                       // 下标赋值（推荐）
[mDict setObject:@"value" forKey:@"key"];        // 等价写法
[mDict removeObjectForKey:@"key"];               // 删除
[mDict removeAllObjects];                        // 清空

// ⚠️ key 不能为 nil（会崩），value 也不能为 nil
// 如果需要表示"空"，用 [NSNull null]
mDict[@"key"] = [NSNull null];  // 占位空值
```

> **C++ 类比**：`NSDictionary` ≈ `std::unordered_map<std::string, void*>`（key 通常是 NSString，value 是任意 id）。

## 4. NSNumber / NSValue — 包装标量

ObjC 的集合（NSArray / NSDictionary）只能存对象（id），不能直接存 int/float/struct。需要用 NSNumber / NSValue 包装。

### NSNumber — 包装标量类型

```objc
NSNumber *intNum = @42;                          // 等价于 [NSNumber numberWithInt:42]
NSNumber *floatNum = @3.14;                      // [NSNumber numberWithFloat:3.14]
NSNumber *boolNum = @YES;                        // [NSNumber numberWithBool:YES]
NSNumber *fromVar = @(someIntVar);               // 包装变量

// 解包
[intNum intValue];       // → 42
[floatNum floatValue];   // → 3.14
[boolNum boolValue];     // → YES
[intNum integerValue];   // → NSInteger（推荐，跨平台安全）

// 比较
[intNum isEqualToNumber:otherNum];              // 值比较
[intNum compare:otherNum];                       // NSOrderedAscending / NSOrderedSame / NSOrderedDescending
```

### NSValue — 包装结构体和指针

```objc
// NSRange（范围：location + length）
NSRange range = NSMakeRange(0, 5);
NSValue *rangeVal = [NSValue valueWithRange:range];
NSRange r = [rangeVal rangeValue];

// CGPoint / CGSize / CGRect（图形结构体）
CGPoint point = CGPointMake(10, 20);
NSValue *pointVal = [NSValue valueWithCGPoint:point];
CGPoint p = [pointVal CGPointValue];

// 任意指针
void *ptr = somePointer;
NSValue *ptrVal = [NSValue valueWithPointer:ptr];
```

## 5. 常用结构体（不是对象，是 C 结构体）

```objc
// NSRange — 范围
NSMakeRange(loc, len)          // 创建
NSMakeRange(0, 5).location     // 访问字段

// CGPoint — 点
CGPointMake(x, y)              // 创建
CGPointZero                    // (0, 0)

// CGSize — 尺寸
CGSizeMake(width, height)
CGSizeZero

// CGRect — 矩形（origin + size）
CGRectMake(x, y, w, h)        // 创建
CGRectZero                     // (0,0,0,0)
rect.origin.x                  // 访问字段
rect.size.width
CGRectGetMinX(rect)            // 左边界
CGRectGetMidY(rect)            // 垂直中点
CGRectIntersectsRect(r1, r2)   // 是否相交
CGRectContainsPoint(rect, pt)  // 是否包含点
```

> 这些不是 ObjC 对象，是纯 C 结构体，存数组/字典里需要用 NSValue 包装。

## 6. nil / Nil / NSNull / NULL 区别

| 值 | 类型 | 含义 | 用在哪 |
|---|---|---|---|
| `nil` | id | 对象指针为空 | `id obj = nil;` `NSString *s = nil;` |
| `Nil` | Class | 类指针为空 | `Class cls = Nil;`（极罕见） |
| `NULL` | void* | C 指针为空 | `int *p = NULL;` C 函数参数 |
| `NSNull` | NSObject | 集合中的空占位 | `arr[0] = [NSNull null]`（表示"有值但值为空"） |

```objc
// ❌ 数组里不能放 nil
[@[@"a", nil, @"b"] ...];  // nil 是数组结束标记，实际只有 @"a"

// ✅ 需要用 NSNull 占位
NSArray *arr = @[@"a", [NSNull null], @"b"];
```

## 7. 类型转换速查

```objc
// NSString ↔ NSNumber
NSNumber *num = @42;
NSString *str = [num stringValue];           // → "42"
NSNumber *back = [NSNumber numberWithInt:[str intValue]];  // → 42

// NSString ↔ NSInteger / int
NSString *s = @"123";
NSInteger n = [s integerValue];               // → 123
NSString *s2 = [NSString stringWithFormat:@"%ld", (long)n];  // → "123"

// NSString ↔ CGFloat / double
CGFloat f = [@"3.14" floatValue];             // → 3.14
NSString *s3 = [NSString stringWithFormat:@"%.2f", f];       // → "3.14"

// NSString ↔ NSURL
NSURL *url = [NSURL URLWithString:@"https://example.com"];
NSString *urlStr = url.absoluteString;

// NSArray ↔ NSString（拼接/分割）
NSString *joined = [@[@"a", @"b"] componentsJoinedByString:@","];  // → "a,b"
NSArray *parts = [@"a,b,c" componentsSeparatedByString:@","];      // → @[@"a",@"b",@"c"]
```

## 关键要点

1. **集合只能存对象**：int/float/struct 用 NSNumber/NSValue 包装
2. **nil 不能放集合**：用 `[NSNull null]` 占位
3. **字符串比较用 `isEqualToString:`**，不要用 `==`（比较的是指针地址）
4. **字面量语法**：`@""` / `@[]` / `@{}` / `@42` 是日常最常用的创建方式
5. **可变 vs 不可变**：`NSArray`(不可变) / `NSMutableArray`(可变)，字典同理——C++ 没有这个区分，这是 ObjC 的设计惯例
